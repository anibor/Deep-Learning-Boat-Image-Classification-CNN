# Venice Boat Classification using CNN

This project applies Convolutional Neural Networks (CNN) to classify boat images into categories such as gondola, motorboat, and ferry.

## Features
- Image preprocessing using ImageDataGenerator
- CNN model built with TensorFlow/Keras
- Model training and evaluation
- Accuracy visualization

## Tools Used
- Python
- TensorFlow / Keras
- Google Colab

## Note
Dataset not included due to size. Connect Google Drive in Colab to access dataset.
